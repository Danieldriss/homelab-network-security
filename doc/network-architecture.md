# Network Architecture

## Descripción General

El laboratorio está dividido en múltiples segmentos de red con el objetivo de simular un entorno realista y aplicar controles de seguridad.

## Segmentación

- LAN: dispositivos personales y administración.
- SERVERS: infraestructura del laboratorio (SIEM, honeypot, DNS, etc.).
- IOT: dispositivos aislados.
- GUEST: red de invitados con acceso restringido.

## Objetivo de la Segmentación

- Reducir superficie de ataque.
- Evitar movimiento lateral.
- Simular arquitectura empresarial básica.
- Preparar el entorno para integración con SIEM.
